# Cinematic Company Portfolio Requirements

## Objective
Create a modern, scroll-driven single page web application used during a university lecture to present the company's work and projects.

## Tech Stack
- Framework: Next.js (React)
- Styling: Vanilla CSS module/global with Design Tokens from Stitch ('Cinematic Engineering' style)
- Animations: Framer Motion

## Core Features
1. **Hero Section**: Logo, Tagline, Parallax Background
2. **Company Work Areas (Services)**: Grid layout presenting AI, Cloud, Data, and Automation categories.
3. **Team & Project Showcase**: Alternating layout for different engineers and projects, demonstrating actual work done.
4. **Final Vision Section**: Closing statement.

## Design
- Color Palette: Deep Blue / Charcoals (`#060E20`, `#091328`), Cyan Accent (`#3BBFFA`)
- Typography: Inter (Headlines), Space Grotesk (Labels)
- Layout: Asymmetric, floating content layers, ghost borders, seamless scrolling transitions.
